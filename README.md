# Wizard's Honey v3.0 - Universal Security Suite

Advanced honeypot and hardening suite for standard web servers and AI proxies.

## Features
- **Dual Modes**: Toggle between `STANDARD` (Web Server) and `PROXY` (9router/AI Proxy) personalities.
- **Deep Packet Inspection**: Monitors AI payloads for prompt injections and key exfiltration.
- **Wizard Shield**: Auto-hardening scripts for Next.js/Express projects.

## Operation
Set `HONEY_MODE` environment variable to `STANDARD` or `PROXY` before running `python app.py`.
